AI TOOLS DESCRIPTION


Chatgpt
This is a chat generative pretrained transformer.
It's capable of carrying on conversations with human users and generating a wide range of text outputs including recipes, computer code, essays and personal letters.
Midjourney
Midjourney is an AI-based image generation tool that uses a text-to-image model.
A User can give text prompt and it creates an image based on it.
AI Sora
This is a generative AI model that takes users prompt and gives generates video accordingly.
InstantID
This is Image generator model that takes our own image as reference.
change the image into different styles maintaining the same identity of us.
Gemini AI
Google Gemini AI provides users with personalized content recommendations based on their historical interactions, searches, and preferences.
Perplexity AI
Perplexity AI is a powerful tool that can help users find information on a wide range of topics.
It is an excellent resource for students, researchers, and anyone else who needs to find accurate and comprehensive answers to their queries.
Tome
This Tome generate presentations if we gave the topic in our prompt.
Trace.zip
This is a AI model that gives the user interface of mobile application that we want to include in mobile.
It gives us code and design. At present, it works on only mac applications and gives us swift UI.
fooocus_colab
This is an Alternative of Midjourney that takes text prompt from the user and generate images.
Suno AI
This Model generate a whole song for some seconds if we just gave the description of the song which we want.
Krea AI
This AI model can take realtime drawings from the user and generate image from it.
It can take shapes and convert them into image.
Hiber 3D
This Generative AI that gives us 3d world by entering the prompt.
Gamma AI
This Generative AI generate webpages and Presentations by entering the prompt.
